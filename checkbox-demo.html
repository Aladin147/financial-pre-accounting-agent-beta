Checkbox Component Demo

Checkbox & CheckboxGroup

Highly customizable checkbox components with support for various states including checked, unchecked, indeterminate, disabled, and error states. The CheckboxGroup component provides functionality for managing groups of related checkboxes with support for select all functionality.

Basic Checkbox Usage

Checkboxes can be used individually or in groups. They support various states like checked, unchecked, indeterminate, disabled, and error states.

Checkbox States

// Basic usage <Checkbox label="Basic Checkbox" name="basic" onChange={handleChange} /> // Indeterminate state <Checkbox label="Select All" indeterminate={someSelected && !allSelected} checked={allSelected} onChange={handleSelectAll} />

Checkbox Sizes

// Size variants <Checkbox label="Small Checkbox" size="small" /> <Checkbox label="Medium Checkbox (Default)" size="medium" /> <Checkbox label="Large Checkbox" size="large" />

CheckboxGroup Component

The CheckboxGroup component manages a group of related checkboxes, handling selection state and providing select all functionality.

Vertical CheckboxGroup

// Vertical CheckboxGroup with select all const [notifications, setNotifications] = useState(['email']); const notificationOptions = [ { value: 'email', label: 'Email notifications' }, { value: 'sms', label: 'SMS notifications' }, { value: 'push', label: 'Push notifications' }, { value: 'browser', label: 'Browser notifications' } ]; <CheckboxGroup name="notifications" label="Notification Preferences" options={notificationOptions} value={notifications} onChange={setNotifications} selectAllOption="Select all notification methods" />
