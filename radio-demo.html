Radio Component Demo

Radio & RadioGroup

Highly customizable radio button components with support for various states including checked, unchecked, disabled, and error states. The RadioGroup component provides functionality for managing related radio options with both vertical and horizontal layouts.

Basic Radio Usage

Radio buttons allow users to select a single option from a set. They follow a mutually exclusive pattern where only one option can be selected at a time.

Radio States

// Basic usage <Radio label="Unchecked Radio" name="demo_radio" value="unchecked" /> // Checked radio <Radio label="Checked Radio" name="demo_radio" value="checked" checked /> // Error and success states <Radio label="Error State" error name="state_radio" value="error" /> <Radio label="Success State" success name="state_radio" value="success" checked />

Radio Sizes

// Size variants <Radio label="Small Radio" size="small" name="size_radio" value="small" /> <Radio label="Medium Radio (Default)" size="medium" name="size_radio" value="medium" checked /> <Radio label="Large Radio" size="large" name="size_radio" value="large" />

RadioGroup Component

The RadioGroup component manages a group of related radio buttons, handling selection state and layout direction.

Vertical RadioGroup

// Vertical RadioGroup const [selectedGender, setSelectedGender] = useState('female'); const genderOptions = [ { value: 'male', label: 'Male' }, { value: 'female', label: 'Female' }, { value: 'other', label: 'Other/Non-binary' } ]; <RadioGroup name="gender" label="Gender" options={genderOptions} value={selectedGender} onChange={setSelectedGender} />

Horizontal RadioGroup

// Horizontal RadioGroup const [selectedPaymentMethod, setSelectedPaymentMethod] = useState('card'); const paymentMethodOptions = [ { value: 'card', label: 'Credit Card' }, { value: 'bank', label: 'Bank Transfer' }, { value: 'paypal', label: 'PayPal' }, { value: 'crypto', label: 'Cryptocurrency' } ]; <RadioGroup name="payment_method" label="Payment Method" options={paymentMethodOptions} value={selectedPaymentMethod} onChange={setSelectedPaymentMethod} direction="horizontal" spacing="lg" />

Integration with FormField

Please select a subscription plan to continue
// Integration with FormField for validation const [selectedPlan, setSelectedPlan] = useState(''); const [formError, setFormError] = useState(true); const planOptions = [ { value: 'basic', label: 'Basic Plan - $9.99/month' }, { value: 'pro', label: 'Pro Plan - $19.99/month' }, { value: 'enterprise', label: 'Enterprise Plan - $49.99/month', disabled: true } ]; const handlePlanChange = (value) => { setSelectedPlan(value); setFormError(value === ''); }; <FormField label="Select a subscription plan" helperText={formError ? "Please select a subscription plan to continue" : "You selected a plan successfully"} error={formError} > <RadioGroup name="plan" value={selectedPlan} onChange={handlePlanChange} options={planOptions} error={formError} /> </FormField>
